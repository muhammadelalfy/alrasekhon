Introduction

Base URL: http://dispatcher.test

This documentation aims to provide all the information you need to work with our API.

As you scroll, you'll see code examples for working with the API in different programming languages in the dark area to the right (or as part of the content on mobile). You can switch the language used with the tabs at the top right (or from the nav menu at the top left on mobile).

Authenticating requests

This API is not authenticated.

Driver Authorization

login

Example request:
curl --request POST \
    "http://dispatcher.test/driver-api/v1/login" \
    --header "Content-Type: application/json" \
    --header "Accept: application/json" \
    --data "{
    \"login\": \"sunt\",
    \"password\": \"}bWRD\\/{EsoH?w\",
    \"device_id\": \"ipsa\",
    \"device_name\": \"nesciunt\",
    \"device_os\": \"libero\",
    \"device_os_version\": 74.7355,
    \"app_version\": 88,
    \"timezone\": \"Africa\\/Conakry\",
    \"fcm_token\": \"sit\"
}"

const url = new URL(
    "http://dispatcher.test/driver-api/v1/login"
);

const headers = {
    "Content-Type": "application/json",
    "Accept": "application/json",
};

let body = {
    "login": "sunt",
    "password": "}bWRD\/{EsoH?w",
    "device_id": "ipsa",
    "device_name": "nesciunt",
    "device_os": "libero",
    "device_os_version": 74.7355,
    "app_version": 88,
    "timezone": "Africa\/Conakry",
    "fcm_token": "sit"
};

fetch(url, {
    method: "POST",
    headers,
    body: JSON.stringify(body),

    --header "Content-Type: multipart/form-data" \

    "http://dispatcher.test/driver-api/v1/files"
);

    "Content-Type": "multipart/form-data",
    "Accept": "application/json",
};

body.append('type', 'driver_avatar');
